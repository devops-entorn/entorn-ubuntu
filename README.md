# Entorn for WSL / Ubuntu 22.04

- Modify ``c:\Users\MyUser\.wslconfig`` adding:

```
[wsl2]
guiApplications=false
```

- Run install-1.sh
- Log out and log in again
- Run install-2.sh
- Run install-kernels.sh

## Adding desktop

- Run ./installers/install-gui.sh

## GUI clients for WSL

- X Server
- VNC client (port 5900)
